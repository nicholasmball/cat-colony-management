Flow & behaviour

  • Reached from colony detail → Record feeding update.
  • Fed defaults on (the common case); big toggle, thumb-reachable.
  • Colony flags (Problem / Food issue / Danger) are optional one-tap chips.
  • Each cat: Seen / Not-seen / Concern — tap-to-toggle, nothing required.
  • Sticky Save (no double-submit); writes 1 feeding_event + N cat_sightings (append-only, offline-idempotent).
  • Success banner back on the colony; events never block on missing data.
  • Any member (incl. Feeder) can record — that's the field job.

a11y: ≥44px targets, state shown by label + colour (never colour alone), visible focus; works at 375px. Detailed sighting semantics are the Cat sightings task.
