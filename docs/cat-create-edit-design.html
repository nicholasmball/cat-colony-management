Behaviour, permissions & a11y

  • Same identifier rule on create and edit (name OR temp id); friendly inline error otherwise.
  • Writes RLS-gated to Caretaker/Admin; edit page also redirects Feeders out server-side.
  • SubmitButton disables on submit (no double-save); revalidatePath refreshes the colony after save.
  • Status (active / missing / …) is not edited here — that's the Missing-cat review flow.
  • Labelled fields, ≥44px targets, visible focus; responsive 375/768/1280.
