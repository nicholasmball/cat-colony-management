Design system & shared UI

Building on the Requirements. The single source of brand truth: tokens in globals.css (@theme), class primitives in lib/ui.ts, reusable components in components/. Matched to the live SCoT public site (streetcatsoftavira.org): warm-white page, near-black text, the brand action colour electric indigo (#170df2), periwinkle secondary accent, Cormorant Garamond display + Raleway body.

Exposed as CSS variables and mapped to Tailwind utilities: bg-surface, text-muted, text-accent, border-border. Accent (#170df2) is the site's --wst-color-action; white-on-accent contrast ≈ 8.8:1 (AAA).

Typography

Cormorant Garamond — display / headings
Raleway — body text, the workhorse for forms and lists.
Muted caption — timestamps, helper text, secondary metadata.

Loaded via next/font/google: --font-display-var (Cormorant Garamond, weights 400/500/600) + Raleway as the base sans — the same pairing as the public site.

pill / status
card

Surface container with border + radius. Used for org cards, colony rows, cat tiles.

Tokens: btnPrimary, btnGhost, card, input, fieldLabel, pill — one definition, applied everywhere.

Reusable components

  • <Logo /> — SCoT mark (dark:invert for contexts that need it).
  • <SubmitButton /> — useFormStatus pending state ("Saving…"), disables to prevent double-submit (fixed the duplicate-colony bug).
  • components/icons.tsx — Home / Paw / Chevron SVGs sharing currentColor.

Accessibility & behaviour notes

  • ≥44px interactive targets; visible focus ring uses the accent colour.
  • Accent on cream and foreground on cream both clear WCAG AA for text.
  • State never conveyed by colour alone — pairs with text/icon.
  • Primitives are class strings, not heavyweight components — zero runtime cost, easy to compose.
